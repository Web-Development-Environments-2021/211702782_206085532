Difficulties we ran into:

It was quite challenging getting used to javascript and the differences between javascript and jquery. Making the navigation between the pages seemed an easy task at first, but along with using a form we ran into bugs on the way that needed our attention.

Making our imagination a reality through CSS and JS was another challenge but it was very rewarding, enjoyable and a great learning experience.

The problems we faced with the logic of the maze where mainly in adjusting the sizing of the maze to match the coordinates for the characters movements and their drawing. Another interesting problem was how to enable different speeds for the monsters and the pacman, not to make the game too difficult for the player. Also a challenge was to enable sound to run automatically by first interacting with the DOM.
